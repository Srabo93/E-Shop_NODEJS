# E-Commerce_NodeJS

Simple E-commerce Shop
e-com.srabodev.de/

### Dev Setup

```

npm run container

npm install

npm run dev

/public/checkout.js -> set /create-order api call to localhost

database seeder: node config/seeder.js

```

.env

```

PORT=
SESSION_SECRET=
TOKEN_SECRET=

SMTP_HOST=
SMTP_PORT=
SMTP_USER=
SMTP_PASSWORD=

FROM_NAME=
FROM_EMAIL=
PSQL_URI=
NODE_ENV=development
```
